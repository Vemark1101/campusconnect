# User Guide

## How to Run

1. Start Apache and MySQL in XAMPP.
2. Import `sql/social_app.sql` in phpMyAdmin.
3. Open `http://localhost/campusconnect/public/index.php`.

## How to Use

### Register

Create an account using full name, username, and password.

### Login

Log in using your account credentials.

### Posts

Create, edit, and delete your own posts. You may also upload an optional image.

### Comments

Add comments to posts and manage your own comments.

### Likes

Like or unlike a post.

### Profile

Update your name, bio, and profile photo.

### Search and Chat

Search users by name or username, then open chat to send a message.
