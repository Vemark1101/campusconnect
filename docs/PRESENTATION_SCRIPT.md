# Presentation Script

## Introduction

Good day. Our final project is CampusConnect, a mini social networking web application built using PHP, MySQL, Bootstrap, JavaScript, and MVC architecture.

## Main Features

- User authentication
- Profile management
- Posts, comments, and likes
- User search
- Direct messaging

## Architecture

- Models handle database access
- Controllers handle routing and business logic
- Views handle the interface
- `public/index.php` works as the main router

## Demo Flow

1. Register a user
2. Log in
3. Open the newsfeed
4. Create a post
5. Edit and delete a post
6. Add, edit, and delete a comment
7. Like and unlike a post
8. Edit profile info and upload a picture
9. Search users
10. Send a direct message

## Security Practices

- Prepared statements
- Password hashing
- Output escaping
- Session-based authentication
- Ownership checks

## Challenges and Learnings

- Applying MVC structure correctly
- Managing sessions and validation
- Handling file uploads and CRUD features
- Making the interface responsive
